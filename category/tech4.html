题目3 : 集合

时间限制:12000ms

单点时限:6000ms

内存限制:256MB

描述

统计满足下列条件的集合对(A, B)的数量：

  • A,B都是{1, 2, …, N}的子集；

  • A,B没有公共的元素；

  • f(A)<= f(B)。f(S)定义为S中所有元素的按位异或和。例如， f({}) = 0, f({1, 3}) = 2。

因为答案可能很大，你只需要求出它除以M的余数。

输入

第一行一个整数T (1 ≤ T ≤ 10)，表示数据组数。
